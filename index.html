What is this?

Typographi.css is a CSS-file containing classes that makes it easier to utilize OpenType font-feature-settings. Such as kerning, ligatures or numeric alternatives.

Keep in mind! When you test these features make sure that you are using a font with OpenType features. Also know that not all character combinations have ligatures, and OpenType fonts will not necessarily share all the features.

This site uses Proxima Nova and Adobe Text Pro. Both of these come with quite a wide array of OpenType options.

Download and try

The CSS file is hosted in Github to enable anyone with interest in web typography to contribute. This will help it remain useful and in sync with time.
